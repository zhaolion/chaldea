## 如何创建自己的迦勒底
- install hugo
- git add submodule {your post repo} content/post
- git add submodule {your github.io repo} dist/public
- git add git@github.com:zhaolion/maupassant-hugo.git themes/lion
- cp config.toml.example config.toml
- ./depoly
